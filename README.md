# Python AI Agent

使用 Python 构建 AI Agent 的学习项目。

## 环境配置

```bash
# 创建并激活 conda 环境
conda create --name python-ai-agent python=3.11 -y
conda activate python-ai-agent
```

## 项目结构

```
python-ai-agent/
├── notes/          # 学习笔记（Markdown）
├── src/            # 源代码
├── pyproject.toml  # 项目配置
└── README.md
```
